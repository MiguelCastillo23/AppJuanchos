## **1. MANUAL DE USUARIO**

### **1.1 Introducción**
Juanchos es una aplicación móvil diseñada para revolucionar la experiencia de hacer pedidos de comida, ofreciendo a los clientes una plataforma intuitiva y completa que simplifica todo el proceso. Con un catálogo digital atractivo y un sistema de recomendaciones personalizadas, la app sugiere platos basándose en tus gustos e historial, haciendo cada vez más fácil descubrir nuevos favoritos. Los usuarios pueden navegar fácilmente por el menú, añadir múltiples productos a su carrito y gestionar sus pedidos de forma eficiente, todo protegido con un inicio de sesión seguro mediante correo electrónico.

La aplicación no solo optimiza el momento del pedido, sino que también enriquece la experiencia general al mantener un historial detallado que permite repetir combos anteriores con un solo clic. Diseñada para ahorrar tiempo y eliminar las complicaciones de las filas o llamadas telefónicas, Juanchos pone el control total en manos del usuario, garantizando comodidad y satisfacción desde la selección hasta la entrega. ¡Bienvenido a una nueva forma de disfrutar de tu comida favorita, donde tus preferencias son siempre la prioridad!
### **1.2 Requisitos del Sistema**
- **Sistema Operativo:** Android 5.0 o superior
- **Memoria RAM:** Mínimo 2GB recomendado
- **Conexión a Internet:** Requerida para realizar los pedidos

### **1.3 Instalación**
**Paso 1:** Descargar el APK
```bash
# Desde el repositorio GitHub
https://github.com/MiguelCastillo23/AppJuanchos.git
```

**Paso 2:** Habilitar instalación de fuentes desconocidas
- Ajustes → Seguridad → Fuentes desconocidas (ACTIVAR)

**Paso 3:** Instalar la aplicación
- Buscar el archivo `app-release.apk`
- Tocar en "Instalar"

### **1.5 Troubleshooting Básico**

#### **Problema: La aplicación no se instala**
**Solución:**
1. Verificar que "Fuentes desconocidas" está activado
2. Asegurarse de que el APK no está corrupto
3. Revisar espacio disponible en el dispositivo

#### **Problema: La aplicación se cierra inesperadamente**
**Solución:**
1. Reiniciar la aplicación
2. Limpiar caché de la aplicación
3. Reinstalar la aplicación

#### **Problema: Error de conexión**
**Solución:**
1. Verificar conexión a Internet
2. Revisar configuración de firewall
3. Reiniciar el router
